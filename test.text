## 📊 NEW ML FEATURE — Geopolitical Risk Score

This adds a **10th feature** to the XGBoost model (Model 1):

```python
def calculate_geopolitical_score(route_waypoints, danger_zones):
    """
    For each waypoint on the route,
    check if it falls inside any
    active danger zone.
    Return composite geopolitical
    risk score 0-100.
    """
    max_severity = 0

    for waypoint in route_waypoints:
        for zone in danger_zones:
            distance = haversine(
                waypoint,
                zone.center
            )
            if distance <= zone.radius_km:
                # Inside danger zone
                zone_score = zone.severity * 10
                if zone.type == "WAR_ZONE":
                    zone_score = min(100, zone_score * 1.5)
                max_severity = max(
                    max_severity,
                    zone_score
                )

    return max_severity  # 0-100
```
